# 🤝 Contribuindo com o Baby Tracking App

Obrigado pelo interesse em contribuir com o **Baby Tracking App**, um projeto open source educacional da **Nova Era Tech School**.

Antes de tudo, é importante entender que este projeto faz parte de uma **formação prática e imersiva**, criada para simular o dia a dia real de um programador profissional.

---

## 🎯 Quem pode contribuir

A contribuição neste projeto é destinada a **alunos da Nova Era Tech School** que:

- Possuem acesso à **área de membros**
- Estão matriculados em alguma formação
- Desejam ganhar **experiência real de mercado** enquanto estudam

> 🔒 Este é um projeto open source com **processo de contribuição educacional guiado**.  
> Por isso, o fluxo de entrada acontece pela plataforma da Nova Era.

---

## 🧭 Visão geral do processo de contribuição

O fluxo de contribuição segue estas etapas:

1. Acesso à plataforma da Nova Era
2. Entrada no projeto Baby Tracking
3. Escolha de uma tarefa no board
4. Estudo direcionado
5. Implementação
6. Comunicação com o time
7. Entrega e feedback

Cada etapa foi pensada para simular o ambiente real de trabalho.

---

## 🗂️ Passo 1 — Acessar o projeto na plataforma

Dentro da **área de membros da Nova Era Tech School**:

1. Acesse a seção **Projetos**
2. Encontre o projeto **Baby Tracking App**
3. Leia a descrição geral do projeto
4. Acesse o **board de tarefas** (Kanban)

---

## 📋 Passo 2 — Escolher uma tarefa

No board do projeto, você encontrará tarefas organizadas por área, como:

- Android
- iOS
- Web
- Back-end
- Dados

👉 **Escolha uma tarefa relacionada à área que você está estudando no momento.**

Cada tarefa contém:
- Descrição clara do que precisa ser feito
- Critérios de aceite
- **Lista de aulas que você deve assistir** para conseguir executar a tarefa

> 💡 Mesmo que você ainda não saiba fazer a tarefa, ela já indica exatamente o que estudar.

---

## 💻 Passo 3 — Preparar o ambiente técnico

Antes de começar a desenvolver, você deve:

1. Fazer o **checkout do repositório**
2. Clonar o projeto para sua máquina
3. Seguir as instruções técnicas específicas da sua área
4. Rodar o projeto localmente

Se algo **não rodar como deveria**, siga as orientações da seção de comunicação abaixo.

---

## 🧠 Passo 4 — Estudar e implementar

Agora é hora de aprender fazendo:

- Assista às aulas indicadas na tarefa
- Entenda o contexto do projeto
- Implemente a solução com calma
- Siga os padrões definidos pelo time
- Documente quando necessário

Este projeto é sobre **aprendizado**, não apenas sobre entrega.

---

## 💬 Comunicação e comunidade

A comunicação acontece em **dois ambientes diferentes**, cada um com seu propósito.

### 💬 Discord — primeiros contatos e alinhamentos

Use o Discord para:
- Se apresentar ao time
- Dizer em qual área você está estudando
- Avisar qual tarefa você pegou
- Tirar dúvidas rápidas de acesso ou onboarding
- Falar diretamente com o **owner do projeto**, se necessário

Sempre que entrar no projeto, faça uma breve apresentação, por exemplo:
- Quem você é
- Qual área estuda
- Qual tarefa está assumindo

---

### 🧠 Plataforma da Nova Era — dúvidas técnicas

Dúvidas técnicas **devem ser feitas dentro da plataforma da Nova Era**, na área de comunidade do projeto.

Isso inclui:
- Dúvidas sobre código
- Decisões técnicas
- Arquitetura
- Implementação
- Melhorias

👉 Ao postar dúvidas técnicas na plataforma:
- Você gera histórico de aprendizado
- Ajuda outros alunos
- **Ganha pontos**, que podem virar bônus dentro da formação

> ⚠️ Evite centralizar dúvidas técnicas apenas no Discord.

---

## 👤 Owner do projeto

Todo projeto da Nova Era possui um **owner**.

O owner é responsável por:
- Ajudar no direcionamento do projeto
- Tirar dúvidas de bloqueio
- Ajudar em problemas que impedem o progresso
- Manter a organização geral do projeto

Caso algo esteja realmente bloqueando seu avanço, você pode:
- Marcar o owner no Discord
- Explicar claramente o problema
- Mostrar o que você já tentou resolver

---

## ✅ Boas práticas ao contribuir

- Leia toda a descrição da tarefa antes de começar
- Não pegue mais tarefas do que consegue executar
- Comunique quando iniciar e quando finalizar
- Peça ajuda quando travar
- Ajude outros alunos sempre que possível
- Lembre-se: este projeto é um ambiente de aprendizado

- Ganhar experiência prática
- Se preparar para o mercado

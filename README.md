# 👶 Baby Tracking App — Nova Era Open Source Project

O **Baby Tracking App** é um projeto **open source** da Escola Nova Era Tech, criado com um objetivo muito claro:

🎯 **proporcionar uma experiência real de trabalho em tecnologia enquanto os alunos ainda estão estudando.**

Mais do que um aplicativo, este projeto é uma **simulação completa do dia a dia de um programador profissional**, envolvendo pessoas reais, usuários reais e processos reais.

---

## 🕵️‍♀️ Sobre do projeto

O Baby Tracking App foi criado para ajudar pais e responsáveis a acompanharem rotinas importantes do bebê, como:

- 🍼 Alimentação  
- 😴 Sono  
- 🛁 Banho  
- 🧸 Outros cuidados do dia a dia  

Mas o **principal objetivo do projeto não é apenas o aplicativo em si**.

Este projeto existe para que os alunos da Nova Era:

- Trabalhem em **um produto real**
- Colaborem em **times multidisciplinares**
- Vivenciem **processos usados por empresas de tecnologia**
- Ganhem **experiência prática de mercado** antes mesmo de se formarem

---

## 🧠 A iniciativa educacional por trás do projeto

Na Nova Era Tech School, acreditamos que **experiência não se estuda, se vive**.

Por isso, este projeto faz parte de uma formação onde os alunos:

- Estudam **enquanto constroem**
- Aprendem **enquanto entregam**
- Evoluem **resolvendo problemas reais**

Aqui, o aluno não apenas assiste aulas — ele **atua como programador**.

---

## 🔁 Simulando o dia a dia de um programador

O desenvolvimento do Baby Tracking App segue práticas reais do mercado, como:

- Metodologia **Scrum**
- Organização com **Kanban**
- Cerimônias como:
  - Daily
  - Planning
  - Review / Demo
  - Retrospectiva

Os alunos aprendem não apenas a programar, mas a:

- Trabalhar em equipe
- Comunicar progresso
- Estimar tarefas
- Lidar com dependências
- Evoluir um produto ao longo do tempo

---

## 🧑‍💻 Como os alunos participam

A experiência é **ponta a ponta**.

Funciona assim:

1. O aluno entra no projeto
2. Escolhe ou recebe uma tarefa real do backlog
3. Mesmo que ainda não saiba executar a tarefa:
   - A própria tarefa indica **quais aulas estudar**
   - Quais conceitos aprender
   - Qual é o objetivo esperado
4. O aluno estuda
5. Aplica o conhecimento no projeto
6. Entrega a tarefa
7. Recebe feedback
8. Evolui como profissional

Isso cria um ciclo contínuo de **estudo + prática + feedback**.

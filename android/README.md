# 👶 Baby Tracking — Android App

Este repositório contém o **aplicativo Android** do **Baby Tracking**, um app criado para ajudar pais e responsáveis a acompanharem a rotina do bebê de forma simples, visual e organizada.

O aplicativo faz parte de um projeto open source da **Nova Era Tech School** e é desenvolvido como um **produto real**, com usuários reais e evolução contínua.

---

## 🎯 Objetivo do aplicativo

O Baby Tracking App tem como objetivo permitir que pais e responsáveis:

- Registrem a rotina diária do bebê
- Acompanhem dados importantes ao longo do tempo
- Visualizem padrões de sono, alimentação e cuidados
- Tenham insights baseados nos dados registrados

Tudo isso através de uma experiência simples, moderna e intuitiva.

---

## ✨ Funcionalidades

- Autenticação de usuários
  - Login
  - Cadastro
  - Gerenciamento de sessão

- Registro de atividades do bebê
  - Alimentação
  - Sono
  - Banho
  - Outros cuidados

- Home com visão geral
  - Ações rápidas para novos registros
  - Histórico resumido dos últimos dias
  - Card de insights com inteligência artificial

- Histórico completo (Tracking)
  - Lista de todos os registros
  - Ordenação por data
  - Diferentes tipos de eventos

- Insights e gráficos
  - Quantidade de leite por dia
  - Horas de sono por dia
  - Evolução ao longo do tempo

- Perfil do bebê
  - Foto
  - Nome
  - Data de nascimento
  - Idade em semanas
  - Peso e altura

- Configurações
  - Editar perfil
  - Notificações
  - Exportação de dados
  - Suporte

- Navegação com Bottom Navigation
  - Home
  - Tracking
  - Insights
  - Profile

## 🧩 Tecnologias utilizadas (Android)

Este projeto utiliza uma stack moderna, alinhada com **requisitos reais de vagas Android**:

- **Kotlin**
- **Jetpack Compose**
- **XML (Views tradicionais)**  
- **MVVM (Model–View–ViewModel)**
- **ViewModel**
- **Kotlin Flow / StateFlow**
- **Retrofit** (chamadas de API)
- **Injeção de dependência**
- **Arquitetura em camadas**
- **Testes unitários**
- **Testes com Jetpack Compose**
- **Gerenciamento de estados de UI**
- **Boas práticas de organização de código**

O projeto foi pensado para evoluir do **básico ao avançado**, permitindo contribuições em diferentes níveis.
